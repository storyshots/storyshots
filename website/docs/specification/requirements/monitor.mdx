# Монитор

<BalancedMetricsTip improves={[Metric.RegressionProtection, Metric.RefactoringAllowance, Metric.Maintainability]} />

Монитор включает основной результат работы UI части приложения — изображение интерфейса.
Относится к секции [*результата*](/specification/requirements/borders#определение-границ).

:::note
Строго говоря, отображение интерфейса на экране пользователя также является сайд-эффектом, как и любая другая операция вывода. Однако для подчёркивания нюансов работы, данная спецификация выделяет UI в отдельный компонент.
:::
